## Overview
This project analyzes customer churn behavior using SQL to identify key factors influencing customer retention and churn rates. The goal is to understand which customer segments are most likely to leave and why.

---

## Objective
To analyze customer data and identify patterns in churn based on contract type, tenure, and monthly charges in order to generate actionable business insights.

---

## Tools Used
- SQL (SQLite)  
- Excel (for optional visualization)  
- Data aggregation and cleaning techniques  

---

## Approach
- Imported and structured customer dataset in SQLite  
- Used SQL aggregations (COUNT, SUM, AVG) for analysis  
- Applied CASE statements to segment customers  
- Grouped data by contract type and tenure to identify trends  

---

## Key Insights
- Month-to-month customers had the highest churn rate  
- Higher monthly charges were associated with increased churn  
- Long-term customers showed significantly better retention rates  

---

## Skills Demonstrated
- Data aggregation (COUNT, SUM, AVG)  
- CASE statements for segmentation  
- GROUP BY analysis  
- Business insight generation from raw data
